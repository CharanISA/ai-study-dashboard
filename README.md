# AI Study Analytics Dashboard

A full-stack productivity dashboard that helps students track study sessions, analyze productivity trends, and receive AI-powered study recommendations.

## Features

- Study session tracking
- Productivity analytics
- Weekly study statistics
- REST API backend
- SQLite database
- AI-generated recommendations

## Tech Stack

- Frontend: HTML, CSS, JavaScript
- Backend: Python Flask
- Database: SQLite
- AI Integration: OpenAI API

## Future Improvements

- User authentication
- Dark mode
- Calendar integration
- Study streak tracking
- Advanced analytics
